# Macro_to_convert_all_parts_is_assenbly_in_sheet_metal_SW
# SolidWorks Sheet Metal Batch Converter

**VBA-макрос для SolidWorks 2024**, автоматически преобразующий все детали сборки в листовой металл с проверкой развёртки.

> 🇷🇺 Разработан для работы с импортированными деталями (STEP/IGES/Parasolid), у которых отсутствует дерево построения.

---

## ✨ Возможности

- 🔄 Автоматический перебор всех деталей сборки (включая подсборки)
- 📐 Интеллектуальный выбор **фиксированной грани** — плоская грань с наибольшей площадью
- 📏 **Точное определение толщины** через расстояние между параллельными гранями (`GetClosestPointOn`)
- 🔧 Два метода преобразования:
  - `InsertConvertToSheetMetal` (7 аргументов) — для **плоских** деталей
  - `InsertConvertToSheetMetal2` (10 аргументов) — для **гнутых** деталей с автопоиском сгибов
- ✅ **Проверка развёртки** — если развёртка не получилась (труба, профиль, сложная деталь), SM автоматически удаляется
- 💾 Автосохранение обработанных деталей
- 🚫 Пропуск дубликатов, подавленных компонентов, ReadOnly-файлов и деталей с уже готовым SM
- 📋 Подробный лог и итоговый отчёт со списком проблемных деталей

---

## 📋 Требования

- **SolidWorks 2024** (SP5.0 — протестировано)
- Может работать на других версиях SW 2020+ (не проверено)
- Макрос запускается на **открытой сборке** (`.sldasm`)
- Детали — импортированные из STEP/IGES/Parasolid (без дерева построения)

## 🚀 Установка и запуск

1. Откройте сборку (`.sldasm`) в SolidWorks
2. **Сделайте резервную копию всех файлов** — макрос перезаписывает детали
3. Перейдите в **Инструменты → Макрос → Создать** (или `Alt+F8` → Создать)
4. Удалите весь код по умолчанию (`Ctrl+A`, `Delete`)
5. Скопируйте содержимое `ConvertToSheetMetal_v9.swp` в редактор VBA
6. Убедитесь, что в **Tools → References** подключены:
   - `SldWorks 20XX Type Library`
   - `SolidWorks 20XX Constant Type Library`
7. Нажмите **F5** для запуска

## ⚙️ Как это работает

```
Сборка (.sldasm)
    │
    ├─ Получение всех компонентов
    │
    ├─ Для каждой уникальной детали (.sldprt):
    │   │
    │   ├─ Открытие детали
    │   ├─ Проверка: уже SM? ReadOnly? → пропуск
    │   │
    │   ├─ Анализ геометрии:
    │   │   ├─ Поиск плоской грани с MAX площадью → фиксированная грань
    │   │   ├─ Подсчёт кривых граней → гнутая/плоская деталь
    │   │   └─ Определение толщины через GetClosestPointOn
    │   │
    │   ├─ Выделение ТОЛЬКО фиксированной грани (Mark = 2)
    │   │   (без рёбер сгиба — SW сам определит сгибы)
    │   │
    │   ├─ Преобразование:
    │   │   ├─ Плоская → InsertConvertToSheetMetal (v1, 7 args)
    │   │   └─ Гнутая  → InsertConvertToSheetMetal2 (v2, 10 args)
    │   │
    │   ├─ Проверка развёртки:
    │   │   ├─ FlatPattern.Unsuppress → ForceRebuild
    │   │   ├─ OK  → свернуть развёртку, сохранить ✅
    │   │   └─ ERR → удалить SM, сохранить чистую деталь ❌
    │   │
    │   └─ Закрытие
    │
    └─ Итоговый отчёт
```

## 📏 Определение толщины

### Основной метод: GetClosestPointOn

Без использования `GoTo`, с явным приведением типов через `CDbl()`:

1. Находим **максимальную плоскую грань** (фиксированная грань)
2. Получаем точку на грани через `GetClosestPointOn(0, 0, 0)`
3. Перебираем **все** параллельные плоские грани (`|dot(N1, N2)| > 0.99`)
4. Для каждой — расстояние через `GetClosestPointOn`
5. **Минимальное расстояние** = толщина

### Fallback: BoundingBox

Если пара граней не найдена:

1. Проекция BBox на нормаль: `|Nx|·dX + |Ny|·dY + |Nz|·dZ`
2. Минимальный размер BBox
3. По умолчанию — 1 мм

## 🔧 Технические детали

### Проблема с API SolidWorks 2024

В SolidWorks 2024 SP5 метод `InsertConvertToSheetMetal` недоступен через раннее связывание (Early Binding). Решение — **позднее связывание** через `CallByName`.

### Сигнатуры методов

**InsertConvertToSheetMetal** (v1, 7 аргументов — плоские детали):

| # | Параметр | Тип | Описание |
|---|----------|-----|----------|
| 1 | `thickness` | Double | Толщина (метры) |
| 2 | `reverseDirection` | Boolean | Направление толщины |
| 3 | `bendRadius` | Double | Радиус сгиба (метры) |
| 4 | `keepBody` | Boolean | Сохранить исходное тело |
| 5 | `enablePart` | Boolean | Активировать деталь |
| 6 | `autoReliefType` | Long | Тип авто-разгрузки (0 = прямоугольный) |
| 7 | `reliefRatio` | Double | Коэффициент разгрузки |

**InsertConvertToSheetMetal2** (v2, 10 аргументов — гнутые детали):

| # | Параметр | Тип | Описание |
|---|----------|-----|----------|
| 1–7 | | | Аналогично v1 |
| 8 | `collectAllBends` | Boolean | Автопоиск сгибов |
| 9 | `???` | Boolean | (определено экспериментально) |
| 10 | `???` | Double | (определено экспериментально) |

### Проверка развёртки

```
FlatPattern.SetSuppression2(swUnSuppressFeature) → ForceRebuild3
│
├─ IsSuppressed = False И GetErrorCode2 = 0 → ✅ Лист
│   └─ Свернуть → Сохранить
│
└─ Иначе → ❌ Не лист (труба, профиль, массив)
    └─ Удалить SM → Сохранить чистую деталь
```

### Важные находки при разработке

| Проблема | Решение |
|----------|---------|
| `Err 438` — метод недоступен | `CallByName` (позднее связывание) |
| `Dim` внутри `GoTo` ломает переменные | Убрать все `GoTo`, использовать `If/End If` |
| `PlaneParams` возвращает мусорные float | `CDbl()` для всех значений из API |
| Рёбра сгиба создают лишние гибы | Выделять **только** фиксированную грань |
| Трубы/профили конвертируются в SM | Проверка развёртки с откатом |

---

## ⚠️ Известные ограничения

- Очень сложные детали (> 200 граней) могут не преобразоваться — SolidWorks не справляется с автоматическим определением сгибов
- **Сделайте резервную копию** всех файлов перед запуском
- Радиус сгиба по умолчанию равен толщине
- Подробный лог работы — в окне **Immediate** (`Ctrl+G` в VBA-редакторе)
